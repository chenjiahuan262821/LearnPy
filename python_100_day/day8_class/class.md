### 1.类和对象

+ 类是对象的蓝图和模板，而对象是类的实例。
+ 一切皆为对象，对象都有属性和行为，每个对象都是独一无二的，而且对象一定属于某个类。
+ 把一大堆拥有共同特征的对象，包括静态特征（属性）和动态特征（行为）都抽取出来堆在一起，就可以定义出一个叫做“类”的东西。

### 2. 定义类class

在python中以class为关键字定义类，在类中通过之前学习过的函数，将对象的动态特征描述出来。
